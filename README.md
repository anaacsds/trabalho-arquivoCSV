# Trabalho Arquivo CSV

## 📖 Sobre o Projeto

Este projeto foi desenvolvido como trabalho prático da disciplina de AED, com o objetivo de realizar a leitura, armazenamento, ordenação e pesquisa de dados provenientes de arquivos CSV.

Os conjuntos de dados utilizados são públicos e foram disponibilizados pelo Datablist:

https://www.datablist.com/learn/csv/download-sample-csv-files

O sistema permite carregar registros de diferentes tamanhos, armazená-los em uma lista flexível e realizar pesquisas utilizando busca binária após a ordenação dos dados.

---

## 🛠️ Funcionalidades

### 📂 Leitura de Arquivos CSV

O usuário pode selecionar arquivos com diferentes quantidades de registros:

* 100 registros
* 1.000 registros
* 10.000 registros
* 100.000 registros

Após a seleção, o sistema realiza a leitura do arquivo e cria objetos correspondentes aos registros encontrados.

### 📋 Lista Flexível (TAD)

Implementação de uma lista capaz de armazenar objetos da entidade escolhida, permitindo:

* Inserção de registros
* Remoção de registros
* Pesquisa de registros
* Impressão dos dados armazenados

### 🔎 Pesquisa

O sistema permite:

1. Escolher um atributo da entidade.
2. Informar um valor para busca.
3. Ordenar os registros.
4. Executar uma busca binária.
5. Exibir todos os registros encontrados.
